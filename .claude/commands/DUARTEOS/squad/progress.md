# Squad: Progresso

Status atual do projeto — onde estamos, o que foi feito, o que vem a seguir.

**Agente lider:** PM (Supreme Orchestrator)
**Motor:** GSD `progress`

## Descricao

Apresenta visao consolidada do estado do projeto: fases completas, fase atual, proximos passos, blockers e decisoes pendentes. O PM usa isso para manter o time alinhado e priorizar.

## Quando usar

- Quando o usuario pergunta "o que esta pendente?"
- No inicio de uma nova sessao de trabalho
- Apos completar uma fase (para saber o proximo passo)
- Quando precisa de visao geral do roadmap

## Como funciona

1. Execute `/gsd:progress`
2. Complemente com status especifico do projeto

## Output esperado

Resumo visual com fases, progresso e proximo passo recomendado.
