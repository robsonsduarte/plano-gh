# Task: Construir UI

## Objetivo

Implementar as paginas e componentes de interface do usuario, integrados com a API do backend.

## Contexto

- Depende da API estar funcional (build-api)
- Usar componentes do design system do projeto
- Garantir responsividade (mobile-first)
- Tratar todos os estados de UI (loading, erro, vazio, sucesso)

## Criterios de Aceite

- [ ] Paginas principais implementadas e navegaveis
- [ ] Integracao com API funcionando (CRUD completo)
- [ ] Estados de loading, erro e vazio tratados
- [ ] Layout responsivo (mobile, tablet, desktop)
- [ ] Formularios com validacao no cliente
- [ ] Acessibilidade basica (labels, focus, contraste)

## Entregaveis

- Paginas da aplicacao
- Componentes reutilizaveis
- Hooks de integracao com API
- Testes de componentes criticos
