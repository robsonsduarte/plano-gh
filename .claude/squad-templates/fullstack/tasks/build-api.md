# Task: Construir API

## Objetivo

Implementar os endpoints de API necessarios para o projeto, com validacao, autenticacao e tratamento de erros.

## Contexto

- Depende do schema de banco de dados (setup-db)
- Endpoints devem seguir padroes REST
- Autenticacao via token/session conforme stack do projeto
- Validacao de input com schema (zod, joi, etc)

## Criterios de Aceite

- [ ] Todos os endpoints CRUD implementados
- [ ] Validacao de input em todos os endpoints
- [ ] Autenticacao/autorizacao funcionando
- [ ] Tratamento de erros com codigos HTTP corretos
- [ ] Testes para endpoints criticos
- [ ] Documentacao basica dos endpoints

## Entregaveis

- Arquivos de rotas/endpoints
- Middleware de autenticacao
- Schemas de validacao
- Testes de API
